== Installation ==

1. Upload the plugin files to `/wp-content/plugins/smo-social/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to SMO Social menu to configure your platforms
4. Add your social media API credentials for each platform
5. Start scheduling and publishing your content!

== Frequently Asked Questions ==

= What platforms are supported? =

SMO Social supports 30+ social media platforms including Facebook, Twitter, Instagram, LinkedIn, TikTok, YouTube, Reddit, Telegram, Discord, and many more. Check the plugin documentation for the complete list.

= Do I need API keys for each platform? =

Yes, you'll need to obtain API credentials for each platform you want to use. Most platforms offer free developer accounts with API access.

= Is the plugin GDPR compliant? =

Yes, SMO Social includes GDPR compliance features and privacy-focused data handling practices.

= Can I schedule posts in advance? =

Absolutely! SMO Social includes a powerful scheduling system that can queue posts for optimal times across different time zones.

= Does it work with AI content generation? =

Yes, the plugin includes AI-powered features for content generation, hashtag optimization, and best-time scheduling.

= What are the new real-time collaboration features? =

SMO Social now includes real-time collaboration tools such as live editing, internal notes, approval workflows, and team notifications for seamless content creation and review processes.

= How does the widgets system work? =

The advanced widgets system provides customizable dashboard components for analytics, post management, team activity, and performance metrics, allowing you to create a personalized dashboard experience.

= Is there a testing suite included? =

Yes, SMO Social includes a comprehensive automated testing suite covering API integrations, dashboard functionality, security measures, and performance testing to ensure reliability and quality.
